# Chating_Site
NODE JS 기반 채팅 사이트

## 사용언어
- MongoDB
- NodeJS
- JaveScript
- HTML

## 기능
- 채팅 기능
  - 기본 채팅
  - Append
  - Is Typing
  - 닉네임(중복 확인)

- Chatbo
-- 출입 알림
-- 접속자 알림
-- 챗봇 이름 알림
-- 메뉴 추천
--- DB에 저장된 레시피를 한식, 일식, 중식을 나누어서 추천가능

- 관리자 모드
-- 비밀번호로 접근
-- 레시피 삽입, 삭제, 수정가능
